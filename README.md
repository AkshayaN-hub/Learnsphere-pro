# 📚 LearnSphere Pro

A modern, responsive, and interactive learning platform developed as the **Final Capstone Project (Task 5)** for the **ApexPlanet Web Development Internship**.

LearnSphere Pro is designed to provide a clean, user-friendly learning experience while demonstrating essential front-end web development concepts using **HTML**, **CSS**, and **JavaScript**.

---

## 🚀 Project Overview

LearnSphere Pro is a multi-page educational website that showcases modern UI/UX design principles and responsive web development techniques. The project focuses on creating an engaging learning platform with interactive features and smooth navigation.

---

## ✨ Features

* 🎨 Modern and professional user interface
* 📱 Fully responsive design (Desktop, Tablet & Mobile)
* 🌙 Dark/Light mode support
* 🔍 Search functionality
* 📚 Course showcase section
* 📝 Notes management using Local Storage
* ✅ Task Manager with Local Storage
* 📊 Interactive dashboard
* 📰 Blog section
* 📞 Contact form with validation
* 📈 Animated statistics and counters
* 🚀 Smooth scrolling and UI animations
* ⬆️ Scroll-to-top button
* 🎯 User-friendly navigation
* ⚡ Optimized performance and clean code

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* Responsive Web Design
* CSS Flexbox & Grid
